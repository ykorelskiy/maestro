## 🛠️ Заметки по работе с Cline

- **Длинные промпты с готовым кодом → Cline лезет делать всё разом.**
  Решение: короткие промпты, разбивать большую задачу на 2-4 микро-прохода.
- **Жёсткие ограничения работают.** Список "НЕ трогай X, Y, Z" в промпте 
  реально удерживает агент в рамках.
- **Escape останавливает Cline в любой момент.** Diff не применяется без Save.
- **New Task для каждого микро-прохода** — чище контекст, меньше путаницы.
- **Act-режим = быстрее, но требует контроля каждого diff.**
  Plan-режим = безопаснее для незнакомых задач, но медленнее.
